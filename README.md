react-demo
====
头创建一个基于 React, webpack, babel 的模板项目
-------
### 项目结构
<pre>
- src
    - index.js
    - App.js
- .babelrc
- index.html
- package.json
- server.js
- webpack.config.js
</pre>
### 1、初始化项目
    首先新建一个名为react-demo的文件夹，然后在命令行下对项目进行初始化<br/>
`npm init '
    之后需要按照提示输入项目的基本信息，按enter键下一步 ，或者运行：<br/>
`npm init -y  //所有的信息按照默认设置 `
